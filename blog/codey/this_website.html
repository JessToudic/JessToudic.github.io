The making of this website


December 2022

  I had been meaning to make a website for a while, to give myself a little home to put my doodles and other projects/thoughts. :) Since I'm a software engineer, I thought it would be a fun project to code from scratch, instead of using a website builder like Squarespace or Wix. My end goal was to have a functioning website, rather than to become particularly proficient at HTML/CSS. With this in mind, I decided that coding alongside a YouTube tutorial was the best way to learn. So that's just what I did! I began by watching this YouTube video by Brian Design below, first copying out the code and then understanding what it did. Following this, I decided what I wanted my website to look like, and adapted it accordingly (with a bit of Googling to supplement what I'd learnt).

  Once the code was written, I needed to get it online! Thankfully, Sam Boyer helped me get a domain name, setup DNS to point the domain name to the server IP address(es) and get my pages onto the server. :) Ta-da: my doodles, projects and thoughts have a little home! üè†
